## 📅 30-Day Roadmap (5 Days a Week)

### 📅 WEEK 01: Python Fundamentals
*   **Day 01:** Introduction to Python (History | What, Why, How)
*   **Day 02:** Python Syntax, Comments, and First Program
*   **Day 03:** Variables & Fundamental Data Types
*   **Day 04:** Operators in Python (Arithmetic, Logical, Comparison)
*   **Day 05:** Conditional Statements (if, elif, else) & Logic Building

### 📅 WEEK 02: Control Flow & Basic Functions
*   **Day 06:** Introduction to Loops (For Loops & range)
*   **Day 07:** While Loops & Loop Control (break, continue)
*   **Day 08:** Intro to Functions & Python Built-in Functions
*   **Day 09:** User-Defined Functions (Parameters & Return Values)
*   **Day 10:** Global Vs Local Variables (Variable Scope)

### 📅 WEEK 03: Functional Power & OOPs Intro
*   **Day 11:** Higher-Order Functions (Concepts & Examples)
*   **Day 12:** Decorators in Python (Adding Power to Functions)
*   **Day 13:** File I/O (Reading from & Writing to Files)
*   **Day 14:** Exception Handling (Errors & Try-Except)
*   **Day 15:** **Introduction to Object-Oriented Programming (OOPs Concepts)**

### 📅 WEEK 04: Advanced OOPs & API Integration
*   **Day 16:** **Implementing OOPs (Classes, Objects, and Constructors)**
*   **Day 17:** **Methods in OOPs (Instance, Class, & Static Methods)**
*   **Day 18:** Introduction to APIs & Requests Module
*   **Day 19:** Practical API Integration & JSON Handling
*   **Day 20:** Introduction to AI APIs & OpenAI API Setup

### 📅 WEEK 05: AI Applications & Web Frameworks
*   **Day 21:** Building AI-powered Scripts with OpenAI
*   **Day 22:** Introduction to Google AI Studio API (Gemini)
*   **Day 23:** Working with Google AI Studio Models
*   **Day 24:** Introduction to FastAPI (Modern Web Framework)
*   **Day 25:** Required and Optional Parameters in FastAPI

### 📅 WEEK 06: Advanced Web & Final Project
*   **Day 26:** FastAPI Routers (Organizing Web Apps)
*   **Day 27:** HTTP Methods (GET, POST, PUT, DELETE)
*   **Day 28:** Final Python Project (Planning & Setup)
*   **Day 29:** Final Project Implementation (Applying OOPs & APIs)
*   **Day 30:** Project Presentation & Final Review
